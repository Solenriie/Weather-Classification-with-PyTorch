# Weather-Classification-with-PyTorch

## Description
- This project involves classifying weather images using deep learning techniques.
- It was developed as part of a master's course on deep learning and computer vision.
- Dataset: https://www.kaggle.com/datasets/jehanbhathena/weather-dataset

## Technologies and Tools
- **Languages**: Python
- **Libraries / Frameworks**: PyTorch, scikit-learn, Pandas, NumPy, Matplotlib, seaborn
